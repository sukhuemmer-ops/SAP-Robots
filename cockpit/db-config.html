🔧 Datenbankpfad konfigurieren

Pfad zur SQLite-Datenbankdatei. Das Verzeichnis muss existieren und beschreibbar sein. Empfohlen: Netzlaufwerk Z:\DB\ für gemeinsame Nutzung.

⚠️ Neustart erforderlich

Die neue Datenbank wird erst nach einem Neustart des Orchestrators aktiv. Bitte starten Sie den Orchestrator neu:

Orchestrator starten.bat

Tipp: Führen Sie vorher die Datenmigration durch, um alle Daten in die neue Datenbank zu kopieren.

🚛 Datenmigration

Backup vor Migration empfohlen!

Kopiert alle Daten aus der aktuellen Datenbank in eine neue Zieldatei. Die Quelldatenbank bleibt unverändert. Nach der Migration den neuen Pfad oben konfigurieren.

1

Zieldatei festlegen

Pfad der neuen (leeren) Datenbankdatei — Verzeichnis muss existieren.

2

Migration starten

Alle Tabellen und Daten werden via SQLite Backup-API übertragen.

3

Neuen Pfad oben speichern & Orchestrator neu starten

Danach wird automatisch die neue Datenbank genutzt.

Die Datei wird angelegt, falls sie noch nicht existiert. Existierende Daten in der Zieldatei werden überschrieben.
