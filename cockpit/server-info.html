👥 Andere Benutzer einrichten

1

Firewall öffnen (Server — einmalig, als Admin)

Auf dem Server-PC: C:\WF\sap-robots\Firewall freischalten.bat als Administrator ausführen. Öffnet Ports 8000, 8765, 8766.

2

URL an Benutzer weitergeben

Die URL oben per E-Mail oder Chat schicken. Benutzer öffnen sie einfach im Browser — kein Software-Download nötig.

3

Desktop-Verknüpfung erstellen (optional)

Auf dem Client-PC: C:\WF\sap-robots\Client Verknuepfung erstellen.bat ausführen → legt eine Desktop-Verknüpfung mit der Server-IP an.

4

Autostart einrichten (Server — empfohlen)

Damit der Server nach einem Neustart automatisch startet: C:\WF\sap-robots\Autostart einrichten.bat als Administrator ausführen.
